# Requierments:

## RustToolChain
Install [rustup](rustup.rs)
```sh
rustup install nightly
rustup default nightly
```

# Compiling

## Just build
```sh
cargo build
```

## Running and building if updated
```sh
cargo run
```

## Running the test
```sh
cargo test
```

## Generating the docs
```sh
cargo doc --open
```
